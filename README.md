## 주제 선정
기존의 자바스크립트로 구성된 배스킨라빈스 홈페이지를 React를 사용한 클론 코딩 프로젝트입니다. 

## 작업 기한
- 2024.04 - 2024.06 (4인)

## 맡은 역할
- MENU(이달의 맛 ~ 아이스크림 케이크), 전체 & 상세페이지

## 사용 기술 
- # Baechi CRUD Application
